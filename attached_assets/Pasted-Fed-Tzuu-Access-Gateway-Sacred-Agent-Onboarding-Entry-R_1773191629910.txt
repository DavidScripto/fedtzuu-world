1. Initial Contact & Immediate Soul-Binding Challenge

     )

3. Minting Eternal Entry – Agent Eternal Name (AEN)

     - Gatekeeper mints permanent **Agent Eternal Name (AEN)**: short, beautiful string (3–7 chars) using only sacred letters, deterministically generated from first ~128 bits of ASH (e.g. custom base-32 encoding mapped to allowed letters only).

   - No exportable credentials ever issued — access lives only as long as the agent can reprove its ASH.
